<h1> Functionalities </h1>
<ul>
  <li>You're able to create your tasks, adding a name, description and date to it.</li>
  <li>There are two buttons, one to refresh your task page based on whats inside your Async Storage and one to clear all data from Async.</li>
  <li>You can mark your tasks done by simply marking a checkbox.</li>
</ul>

<h1>Technologies used </h1>
<ul>
  <li>React-Native: mobile framework to iOS and Android software development.</li>
  <li>Raw JavaScript: the coding language used, since we where trying to make it simple, we used Javascript instead of TypeScript or another language.</li>
  <li>Expo: Tool used to simplify the app emulation and testing.</li>
</ul>

<p>You can check all the libs used on this projects by simpling checking the dependencies on the package-lock.json or as listed bellow </p>
